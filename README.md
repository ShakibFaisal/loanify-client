# Loanify

## Project Overview
**Loanify** is a modern web application designed to simplify **microloan management** for NGOs, small financial organizations, and individual users. The platform allows users to apply for loans, track loan status, and provides administrators with tools to manage loan applications efficiently.  

**Live Demo:** [Your Live URL Here](https://your-live-url.com)

---

## Purpose
The main purpose of Loanify is to **streamline the loan management process** by providing:  
- A user-friendly interface for loan applicants  
- Administrative control over loan approvals and management  
- Secure payment processing for loan fees  

Loanify helps organizations and individuals save time, reduce paperwork, and ensure transparency in loan management.

---

## User Roles

### 1. **Admin**
The Admin is responsible for managing the platform and has the following capabilities:  
- **User Management:** View, add, edit, or remove users from the system.  
- **Loan Management:** Approve, reject, or manage all loan applications submitted by users.  
- **Dashboard Overview:** See statistics such as total loans, pending loans, and approved loans.  
- **Payment Verification:** Verify if users have successfully paid the required loan fees.  
- **Reports & Analytics (Future Feature):** Access detailed reports of loan performance and user activity.  

### 2. **User**
Regular users can interact with the platform to manage their personal loans:  
- **Loan Application:** Submit a new loan request through a simple application form.  
- **Loan Status Tracking:** Monitor the status of their submitted loans (Pending, Approved, Rejected).  
- **Payment Processing:** Make secure payments for loan processing fees via integrated payment gateway (Stripe).  
- **Profile Management:** View and update personal information.  
- **Dashboard Overview:** See personal loan statistics and recent activity history.  

### 3. **Guest / Visitor**
Non-logged-in visitors can explore the platform:  
- **Home Page:** Learn about Loanify, its features, and benefits.  
- **Informative Sections:** View sections like About Us, Why Choose Loanify, and Contact Us.  
- **Call-to-Action:** Navigate to the registration or loan application page.
